Overview

CKIdValueWrapper allows to easily wrap a value type like a stuct in an NSObject subclass. This type should be interacted with through its C API.

– initWithValue:assigner:releaser:comparator:

Shouldn’t be used directly, see CKIdValueWrapperCreate()
