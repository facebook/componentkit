CKTextKitEntityAttribute Class Reference

Inherits from NSObject
Declared in CKTextKitEntityAttribute.h

Overview

The object that should be embedded with CKTextKitEntityAttributeName. Please note that the entity you provide MUST implement a proper hash and isEqual function or your application performance will grind to a halt due to NSMutableAttributedString’s usage of a global hash table of all attributes. This means the entity should NOT be a Foundation Collection (NSArray, NSDictionary, NSSet, etc.) since their hash function is a simple count of the values in the collection, which causes pathological performance problems deep inside NSAttributedString’s implementation.
