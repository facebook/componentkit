CKTextKitRendererMeasureOption

Measure options are used to specify which type of line height measurement to use.

ASTextNodeRendererMeasureOptionLineHeight is faster and will give the height from the baseline to the next line.

ASTextNodeRendererMeasureOptionCapHeight is a more nuanced measure of the glyphs in the given range that attempts to produce a visually balanced rectangle above and below the glyphs to produce nice looking text highlights.

ASTextNodeRendererMeasureOptionBlock uses the cap height option to generate each glyph index, but combines all but the first and last line rect into a single block. Looks nice for multiline selection.
